## 📖 О проекте

Автоматически собранные VLESS конфигурации от различных провайдеров. Все конфиги находятся в одном файле и регулярно обновляются.

### ✨ Особенности

- 🔄 **Автоматическое обновление** - конфиги обновляются регулярно
- ✅ **Актуальные конфиги** - только рабочие серверы
- 🌍 **Различные локации** - серверы по всему миру
- 📱 **Универсальная совместимость** - работает со всеми V2Ray клиентами

## ⚙️ Настройка

### Метод 1: Импорт одного конфига

**v2rayN (Windows) / v2rayNG (Android)**

1. Скопируйте строку `vless://...` из [all_configs.txt](all_configs.txt)
2. Откройте приложение
3. Нажмите `Ctrl + V` (Windows) или кнопку ➕ → Import (Android)
4. Конфиг автоматически добавится

**Nekoray (Linux/Windows)**

1. Скопируйте строку конфига
2. Откройте Nekoray
3. Клик правой кнопкой → Add profile from clipboard

### Метод 2: Импорт всех конфигов сразу

**Для продвинутых пользователей:**

```bash
# Linux/macOS - скопировать все конфиги
cat all_configs.txt | xclip -selection clipboard

# Windows PowerShell
Get-Content all_configs.txt | Set-Clipboard
```

Затем в клиенте используйте функцию массового импорта.

## 💡 Советы по использованию

### 🎯 Как выбрать лучший сервер?

1. **Импортируйте несколько конфигов** (5-10 штук)
2. **Протестируйте скорость** (в клиенте обычно есть Real Ping или Speed Test)
3. **Выберите самый быстрый** с наименьшим пингом
4. **Сохраните в избранное** для быстрого доступа

### 📊 Понимание параметров

Конфиг выглядит так:
```
vless://uuid@server:port?параметры#название
```

- `server:port` - адрес и порт сервера
- Ищите серверы с **низким пингом** (<100ms)
- **TLS** обеспечивает лучшую безопасность
- **WS (WebSocket)** работает стабильнее за файрволлами
